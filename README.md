# Long Note for taking long notes (Android App With React Native)

## Info

- React Native 0.74.3
- React 18.2.0
- react-native-paper 5.12.3
- @shopify/flash-list 1.7.0

### how to run

- install modules

```shell
npm i
```

- run android (need android studio, virtual machine or android phone)

```
npm run android
```
